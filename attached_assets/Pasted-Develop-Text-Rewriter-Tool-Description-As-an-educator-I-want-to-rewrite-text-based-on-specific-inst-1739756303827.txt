Develop Text Rewriter Tool
Description
As an educator, I want to rewrite text based on specific instructions (e.g., simplify, summarize, or rephrase) so that I can adapt content for different reading levels, purposes, or contexts. This tool will help streamline the process of tailoring written material for diverse educational needs.

The tool should allow users to:

Input original text or upload files to be rewritten.
Provide rewriting instructions (e.g., simplify, summarize, rephrase).
Accept file uploads (one file per prompt) in supported formats:
Upload: CSV, PDF, DOCX, PPT, Plain Text
URLs: YouTube Video, Website, Google Sheets
Process the uploaded content and generate rewritten text tailored to the instructions.
Export the rewritten text in user-friendly formats such as plain text, DOCX, or PDF.
